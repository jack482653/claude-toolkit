# Installation Guide

## Skills (for Claude.ai)

### Method 1: Copy-Paste (Recommended)

1. Open the skill's `SKILL.md` file
2. Copy its entire contents
3. In Claude.ai, paste into project knowledge or conversation

### Method 2: Upload as Project Knowledge

1. Download the `SKILL.md` file
2. Go to your Claude.ai project settings
3. Upload the file to project knowledge

## Commands (for Claude Code)

1. Copy the command `.md` file (e.g., `gc.md`)
2. Place it in your Claude Code commands directory
3. The command will be available via `/commandname`

## Troubleshooting

**Skill not working?**
- Verify content was copied correctly
- Try mentioning the tool name explicitly in your question

**Command not found?**
- Check the file is in the correct directory
- Restart Claude Code
