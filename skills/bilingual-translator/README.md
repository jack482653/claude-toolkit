# Bilingual Translator

Translate English articles into Traditional Chinese with side-by-side HTML output.

## Features

- Dual-column layout (English left, Chinese right)
- Responsive design for mobile
- Source URL reference support

## Usage

Trigger phrases:
- "翻譯這篇文章" with URL
- "雙語對照翻譯"
- "bilingual translation"

## Output

Creates an HTML file with professional styling and paragraph-by-paragraph correspondence.
